## Details of Use

This site is programmatically generated using my simple [Card Lookup Helper](https://gitlab.com/Thoughtscript/card_lookup_helper) tool through the command:

```bash
bash run.sh
```

Images used on this page are retrieved from the [Official PSA](https://www.psacard.com/cert) and the [Official CGC](https://www.cgccards.com/certlookup/) Certificate Population public registries!
